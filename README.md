# CP322 Term Project

Machine Learning project for CP322 focusing on regression tasks on tabular data.


Datasets and predicted feature
1. **Energy Efficiency Dataset** - Predicting Heating Load and Cooling Load
2. **Energy Appliances Dataset** - Predicting Appliances energy consumption


Before Running main.py make sure to install dependencies
```bash
pip install -r requirements.txt
```

run the main.py file

```bash
python main.py
```

 `requirements.txt` for list of all dependencies
- pandas, numpy
- scikit-learn
- xgboost
- pytorch-tabnet
- optuna
- shap
- matplotlib
